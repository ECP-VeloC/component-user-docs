getter functions

the get functions all return KVTREE_FAILURE if any of the following hold:

  • the input hash or the key is NULL
  • the hash has no entry for the specified key
  • the specified key does not have a value otherwise, the function returns KVTREE_SUCCESS and the key value is translated into the appropriate type and stored in the specified buffer
