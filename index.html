Component User Documentation

This site documents user-facing APIs for each of ECP-VELOC components. This documentation comes in the form of header files. Most components can be used with only a single #include of the main header file. However, for some components, additional functionality and utilities are provided in other headers.

These components were originally designed as part of the Scalable Checkpoint/Restart (SCR) library. For more details, read the veloc user documentation.
